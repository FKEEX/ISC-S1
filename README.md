GLOSARIO
1.Como se define un sistema de computo?
  Un sistema de computo se divide en varios componentes que reciben datos, los procesasn y producen información útil, tambien por componen
  de diferentes componenetes, por ejemplo, el hadware + saftware + datos + usuarios + prodecimientos.
  
  2.Como se puede componer un sistema?
  Un sistema puede ser: PC, Movil, servidor, un servicio en la nube o un sistema embebido.
  
3.Cuantos modelos basicos existen de entrada?
  Unos ejemplos pueden ser un teclado, un mouse, una camará de video, sensores, bocinas, 
  Universal Serial Bus incluso Archivos, bases de datos APIs y mensajes de red.
  
4.Cunatos modelos basicos de salida existen?
  Existen varios modelos de salida por ejemplo, Pantallas, audio, impresoras, respuestas web, apps entre otros.
  
5.Como se compone un hadware en una pc?
  Se componen de varios modulos por ejemplo, la CPU, el cerebro principal de una computadora, la memoria, los dicos, 
  donde se almacena toda la información.
  
6.Como se compone un Software?
  Un Software se compone principalmente de un Sistema operativo que se encargar de gestinar el Hardware, tambien algunas
  aplicaciones, y utilitarios, Firmware y donde esta la BIOS.
  
7.Como se compone un Servicio?
  Se compone principalmete por la BIos/ UEFI microcontroladores
  la nube ( IaaS/ PaaS/ SaaS).
  
8.Que son los contenedores en la computacion actual?
  Tiene varios nombres donde el principal llamado dockers, o backups, que almacenan todo tipo de información.
  
9.Que es la ciberseguridad?
  Confidencialidad: Asegura que solo usuarios autorizados accedan a los datos, tambien maneja la seguridad de las contraseñas, 
  phishing y cifrado.
  
10.Que es un Hadware?
  El hardware se refiere a los componentes físicos y tangibles de un sistema informático, tales como la CPU, 
  memoria RAM, discos duros, monitor, teclado y ratón, que permiten el funcionamiento del software.
  
11.Qué es un software?
  Conjunto de componentes lógicos, programas, datos y reglas informáticas intangibles que indican al hardware 
  (componentes físicos) de una computadora, teléfono o dispositivo electrónico qué tareas ejecutar y cómo funcionar.
  
12.que es un servidor en la nube?
  Es un recurso virtualizado y remoto, accesible vía internet, que proporciona capacidad de cómputo,
  almacenamiento y aplicaciones bajo demanda.
  
13.Que es sistema embebido?
  Dispositivo de computación especializado, basado en microcontroladores o microprocesadores, diseñado para
  realizar funciones específicas y dedicadas dentro de un sistema más grande.
  
14.Qué es la IA?
  Es un campo de la informática que desarrolla sistemas capaces de realizar tareas que 
  requieren inteligencia humana, como el aprendizaje, el razonamiento, la percepción y la toma de decisiones autónoma.
  
15.Qué es el Internet?
  Red informática mundial, descentralizada, formada por la conexión directa
  entre computadoras mediante un protocolo especial de comunicación.
